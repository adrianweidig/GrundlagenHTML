Abschnitt 1: Ungeordnete Liste (ul)

Die ul-Liste wird verwendet, um eine ungeordnete Liste zu erstellen:

  • Element 1
  • Element 2
  • Element 3

Abschnitt 2: Geordnete Liste (ol)

Die ol-Liste wird verwendet, um eine geordnete Liste mit numerierten Elementen zu erstellen:

  1. Element A
  2. Element B
  3. Element C

Abschnitt 3: Definitionliste (dl, dt, dd)

Die dl-Liste wird verwendet, um eine Definitionliste zu erstellen:

Begriff 1
Definition 1
Begriff 2
Definition 2

Abschnitt 4: Eingebettete Liste

Listen können auch ineinander verschachtelt sein:

  • Hauptpunkt 1
    • Unterpunkt A
    • Unterpunkt B
  • Hauptpunkt 2

Abschnitt 5: Listen mit unterschiedlichen Markern

Der Markerstil kann durch das type-Attribut geändert werden:

  • Rund
  • Rund
  • Rund
  1. Alpha
  2. Alpha
  3. Alpha

Abschnitt 6: Gemischte Liste

Sie können unterschiedliche Arten von Listen innerhalb derselben Liste mischen:

  • Ungeordnetes Element
  • Geordnetes Element
    1. Unterpunkt 1
    2. Unterpunkt 2
  • Ungeordnetes Element

Abschnitt 7: Beschreibende Liste

Eine beschreibende Liste kann für kurze Erläuterungen verwendet werden:

Begriff 1
Definition 1
Begriff 2
Definition 2

Abschnitt 8: Einfache Checkliste

Sie können Listen für einfache Checklisten verwenden:

  • Aufgabe 1
  • Aufgabe 2
  • Aufgabe 3

Abschnitt 9: Listen mit Verlinkung

Listenelemente können auch Links enthalten:

  • Link zu Beispiel.com
  • Link zu Beispiel.org
